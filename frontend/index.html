Docs
Quickstart for self-hosted deployment and SDK use.
1) Bootstrap an org + API key
Server side (admin)
Set `BOOTSTRAP_TOKEN` on the server and call `POST /admin/bootstrap` with `X-Bootstrap-Token`.\n Copy the returned `api_key` into Settings.\n
2) Send telemetry
Agent/SDK
Use the Python SDK in `sdk/python` to send `prompt_sent` events to `POST /ingest/events`.\n The server will create detections + incidents.\n
3) Investigate
Console
Use Overview, Live Events, Detections, and Incidents. Filter by risk/action and follow trace ids.\n
